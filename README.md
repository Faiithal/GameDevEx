# GameDevEX
A trustless commission hub and credential registry for SEA Game Developers.

## Problem & Solution
SEA game developers struggle with high fees and "trust gaps" when working with global recruiters. GameDevEX uses Soroban smart contracts to provide a tamper-proof "Proof of Skill" registry and automated XLM escrows, ensuring developers get paid instantly upon milestone completion without intermediary delays.

## Suggested Timeline (MVP)
Week 1: Smart Contract Core (Credentialing & Escrow logic).

Week 2: Frontend integration (React/Tailwind) with Stellar Freight/Wallet.

Week 3: Testnet deployment and recruiter dashboard beta.

## Stellar Features Used
Soroban Contracts: Logic for task registration and verification.

XLM Transfers: Instant global payments.

Events: On-chain logging for recruiter transparency.

## Prerequisites
Rust Toolchain (rustup target add wasm32-unknown-unknown)

Soroban CLI v20.0.0+
